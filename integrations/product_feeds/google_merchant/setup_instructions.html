{% trans "Google Merchant Center Setup" %}

{% trans "Follow these steps to connect your store to Google Merchant Center" %}

1

{% trans "Create a Merchant Center Account" %}

{% trans "If you don't have a Google Merchant Center account, create one:" %}

  1. {% trans "Go to" %} Google Merchant Center
  2. {% trans "Sign in with your Google account" %}
  3. {% trans "Follow the setup wizard to create your merchant account" %}
  4. {% trans "Verify and claim your website URL" %}
{% trans "Your Merchant ID is displayed in the top-left corner of Merchant Center (e.g., 123456789)" %}
2

{% trans "Create a Service Account" %}

{% trans "You need a Google Cloud service account to authenticate API requests:" %}

  1. {% trans "Go to the" %} Google Cloud Console Credentials page
  2. {% trans "Select or create a project" %}
  3. {% trans "Click 'Create Credentials' → 'Service Account'" %}
  4. {% trans "Give it a name like 'Spwig Product Feed'" %}
  5. {% trans "Click 'Create and Continue', then 'Done'" %}
3

{% trans "Generate and Download JSON Key" %}

  1. {% trans "Find your service account in the credentials list" %}
  2. {% trans "Click on the service account email" %}
  3. {% trans "Go to the 'Keys' tab" %}
  4. {% trans "Click 'Add Key' → 'Create new key'" %}
  5. {% trans "Select 'JSON' format and click 'Create'" %}
  6. {% trans "The JSON key file will download automatically" %}
{% trans "Keep this JSON key secure! It provides access to your Merchant Center account." %}
4

{% trans "Enable the Content API" %}

  1. {% trans "In Google Cloud Console, go to" %} API Library
  2. {% trans "Search for 'Content API for Shopping'" %}
  3. {% trans "Click on it and then click 'Enable'" %}
5

{% trans "Grant Access in Merchant Center" %}

  1. {% trans "Go to" %} Merchant Center → Settings → Users
  2. {% trans "Click 'Add user'" %}
  3. {% trans "Enter your service account email (from the JSON key)" %}
  4. {% trans "Select 'Standard' or 'Admin' access level" %}
  5. {% trans "Click 'Add user'" %}
{% trans "The service account email looks like: your-service-account@your-project.iam.gserviceaccount.com" %}
6

{% trans "Enter Your Credentials" %}

{% trans "On the next step, you'll need:" %}

  • {% trans "Merchant ID" %}: {% trans "Your numeric Merchant Center ID" %}
  • {% trans "Service Account JSON" %}: {% trans "The entire contents of the downloaded JSON key file" %}
  • {% trans "Target Country" %}: {% trans "The primary country where you sell products" %}
  • {% trans "Content Language" %}: {% trans "The language of your product data" %}
