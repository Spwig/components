# Changelog

## 1.0.0 (2026-02-09)

- Initial release with Revolut Merchant API support
- Card payments (Visa, Mastercard, Amex, Discover, Diners, JCB)
- Digital wallet support (Apple Pay, Google Pay, Revolut Pay)
- Pay by Bank via Open Banking
- Hosted checkout page and integrated checkout widget
- Manual and automatic capture modes
- Full and partial refund support
- Webhook integration with HMAC-SHA256 verification
- 3D Secure authentication
- Multi-currency support (21 currencies)
