Gmail / Google Workspace DNS

{% trans "Important:" %} {% trans "If you're using Gmail's SMTP server, Google automatically signs emails with DKIM. You don't need to configure DKIM yourself. However, SPF and DMARC are still recommended." %}

1 {% trans "SPF Configuration" %}

{% trans "Add this SPF record to authorize Gmail to send emails from your domain:" %}

{% trans "Record Type:" %} TXT | {% trans "Hostname:" %} @ ({% trans "or" %} {{ domain }})
v=spf1 include:_spf.google.com ~all
{% trans "Important:" %} {% trans "If you already have an SPF record, add" %} include:_spf.google.com {% trans "to it. Do not create multiple SPF records - only one SPF record is allowed per domain." %}

{% trans "Example with multiple senders:" %}

v=spf1 include:_spf.google.com include:spf.otherprovider.com ~all

2 {% trans "DKIM Configuration" %}

{% trans "Google automatically signs emails with DKIM when you use Gmail SMTP. No additional configuration is needed." %}

{% trans "For Google Workspace users:" %} {% trans "You can generate a custom DKIM key in the Google Admin Console:" %}
  1. {% trans "Go to" %} Apps → Google Workspace → Gmail → Authenticate email
  2. {% trans "Click" %} Generate new record
  3. {% trans "Copy the TXT record and add it to your DNS" %}

3 {% trans "DMARC Configuration (Recommended)" %}

{% trans "DMARC builds on SPF and DKIM to protect your domain from spoofing:" %}

{% trans "Record Type:" %} TXT | {% trans "Hostname:" %} _dmarc.{{ domain }}
v=DMARC1; p=quarantine; rua=mailto:dmarc@{{ domain }}; pct=100

{% trans "DMARC Policy Options:" %}

  • p=none - {% trans "Monitor only (recommended for testing)" %}
  • p=quarantine - {% trans "Send suspicious emails to spam" %}
  • p=reject - {% trans "Reject unauthorized emails completely" %}

{% trans "Generic SMTP Server DNS Configuration" %}

{% trans "For other SMTP providers, you'll typically need to configure SPF, DKIM, and DMARC records. Contact your SMTP provider for specific DNS record values." %}

1 {% trans "SPF Configuration" %}

{% trans "SPF (Sender Policy Framework) authorizes your SMTP server to send emails from your domain:" %}

{% trans "Record Type:" %} TXT | {% trans "Hostname:" %} @ ({% trans "or" %} {{ domain }})
v=spf1 include:_spf.yourprovider.com ~all
{% trans "Important:" %} {% trans "Replace" %} _spf.yourprovider.com {% trans "with the SPF include value from your SMTP provider." %}

{% trans "Common SPF Mechanisms:" %}

  • include: - {% trans "Include SPF records from another domain" %}
  • a - {% trans "Authorize the domain's A record IP" %}
  • mx - {% trans "Authorize the domain's MX servers" %}
  • ip4: - {% trans "Authorize specific IPv4 address" %}
  • ip6: - {% trans "Authorize specific IPv6 address" %}

{% trans "SPF Qualifiers:" %}

  • ~all - {% trans "Soft fail (recommended) - mark as spam" %}
  • -all - {% trans "Hard fail - reject unauthorized emails" %}
  • ?all - {% trans "Neutral - accept but don't verify" %}

2 {% trans "DKIM Configuration" %}

{% trans "DKIM (DomainKeys Identified Mail) adds a cryptographic signature to emails." %}

{% trans "Your SMTP provider should provide DKIM public key records. These are typically TXT or CNAME records:" %}

{% trans "Record Type:" %} TXT | {% trans "Hostname:" %} default._domainkey.{{ domain }}
v=DKIM1; k=rsa; p=MIGfMA0GCSqGSIb3DQEBAQUAA4GNADCBiQKBgQ...

{% trans "Or as a CNAME:" %}

{% trans "Record Type:" %} CNAME | {% trans "Hostname:" %} default._domainkey
default._domainkey.yourprovider.com
{% trans "Note:" %} {% trans "Contact your SMTP provider for the exact DKIM selector and public key values." %}

3 {% trans "DMARC Configuration" %}

{% trans "DMARC (Domain-based Message Authentication, Reporting, and Conformance) builds on SPF and DKIM:" %}

{% trans "Record Type:" %} TXT | {% trans "Hostname:" %} _dmarc.{{ domain }}
v=DMARC1; p=quarantine; rua=mailto:dmarc@{{ domain }}; pct=100

{% trans "DMARC Parameters:" %}

  • v=DMARC1 - {% trans "Version (required)" %}
  • p= - {% trans "Policy for failed authentication (none, quarantine, reject)" %}
  • rua= - {% trans "Email address for aggregate reports" %}
  • ruf= - {% trans "Email address for forensic reports (optional)" %}
  • pct= - {% trans "Percentage of emails to apply policy to (0-100)" %}
  • sp= - {% trans "Policy for subdomains (optional)" %}
