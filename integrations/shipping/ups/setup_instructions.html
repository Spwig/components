{% trans "Getting Your UPS API Credentials" %}

{% trans "Follow these steps to set up your UPS developer account and obtain API credentials" %}

1

{% trans "Create a UPS Developer Account" %}

{% trans "Visit the UPS Developer Portal and register for a free developer account." %}

{% trans "Open UPS Developer Portal" %}

{% trans "You'll need a valid email address and basic business information to register." %}

2

{% trans "Create a New Application" %}

{% trans "After logging in to your UPS developer account:" %}

  • {% trans "Navigate to 'My Apps' or 'Applications'" %}
  • {% trans "Click 'Create App' or 'Add Application'" %}
  • {% trans "Give your application a descriptive name (e.g., 'My Store Shipping')" %}
  • {% trans "Provide a description for your integration" %}
3

{% trans "Request API Access" %}

{% trans "You need to request access to specific UPS APIs:" %}

  • {% trans "Rating API" %} - {% trans "For calculating shipping rates" %}
  • {% trans "Shipping API" %} - {% trans "For generating shipping labels" %}
  • {% trans "Tracking API" %} - {% trans "For real-time tracking updates" %}
  • {% trans "Address Validation" %} - {% trans "Optional but recommended" %}

{% trans "API access requests may require approval from UPS. This can take 1-2 business days." %}

4

{% trans "Generate OAuth Credentials" %}

{% trans "Once your application is approved and API access granted:" %}

  • {% trans "Open your application in the UPS Developer Portal" %}
  • {% trans "Navigate to the 'Credentials' or 'API Keys' section" %}
  • {% trans "Click 'Generate Client Credentials' or 'Create Credentials'" %}
  • {% trans "You will receive a Client ID and Client Secret" %}
  • {% trans "Copy both values immediately - you'll enter them in the next step" %}
{% trans "Important!" %} {% trans "UPS will only show your Client Secret once. Make sure to save it securely before closing the page." %}
5

{% trans "Choose Your Environment" %}

{% trans "UPS provides two environments for API integration:" %}

  • {% trans "Test Environment (Customer Integration Environment - CIE)" %}: {% trans "Use this for development and testing. It simulates the UPS system without creating real shipments or charges. Highly recommended for initial setup and testing." %}
  • {% trans "Production Environment" %}: {% trans "Use this for live shipments with real tracking numbers and charges. Your UPS account will be billed for actual shipments." %}

{% trans "Start with the Test environment to verify your integration works correctly before switching to Production." %}

6

{% trans "Get Your UPS Account Number (Optional)" %}

{% trans "Your UPS account number is a 6-character alphanumeric identifier:" %}

  • {% trans "Required for generating shipping labels" %}
  • {% trans "Optional for getting shipping rates" %}
  • {% trans "Found on your UPS invoices or in your UPS account dashboard" %}
  • {% trans "Format: 6 characters (e.g., A1B2C3, 123ABC)" %}

{% trans "If you don't have a UPS account number yet, you can still get rates. You'll need it when you're ready to generate shipping labels." %}
