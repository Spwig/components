Wise API Setup Guide

Follow these steps to connect your Wise Business account for affiliate payouts

! Prerequisites

  • A Wise Business account (personal accounts have limited API access)
  • Completed business verification with Wise
  • Multi-currency account activated
  • Sufficient balance in your sending currency
1

Sign Up for Wise Business

If you don't have a Wise Business account:

  1. Go to wise.com/business
  2. Click "Get started" or "Open a business account"
  3. Complete the registration with your business details
  4. Verify your identity and business documentation
  5. Activate your multi-currency account
Note: Business verification typically takes 1-3 business days.
2

Enable Two-Factor Authentication

2FA is required for API access:

  1. Log into your Wise Business account
  2. Go to Settings (gear icon)
  3. Navigate to Security
  4. Enable Two-factor authentication
  5. Set up using an authenticator app (Google Authenticator, Authy, etc.)
3

Generate API Token

Create an API token for integration:

  1. In your Wise account, go to Settings
  2. Click on API tokens
  3. Click "Add new token"
  4. Enter a name (e.g., "Spwig Payouts")
  5. Select "Full access" permission level
  6. Complete 2FA verification
  7. Copy and securely store your API token
Important: Your API token is shown only once. Store it securely - you cannot retrieve it later. If lost, you must generate a new token.
4

Find Your Profile ID (Optional)

The profile ID can be auto-detected, but you can also find it manually:

  1. In your Wise account, go to Settings
  2. Scroll to Your profile ID
  3. Copy the numeric ID

Alternatively, your profile ID is visible in the URL when logged in:
https://wise.com/balances/12345678

5

Set Up Webhooks (Optional)

Webhooks notify your store when transfer statuses change:

  1. In Settings, go to Webhooks
  2. Click "Create new webhook"
  3. Enter your webhook URL:
    https://yourdomain.com/payout-providers/webhooks/wise/
  4. Select events to subscribe to:
    • transfers#state-change - Transfer status updates
    • transfers#active-cases - Issues requiring attention
    • balances#credit - Balance credited (optional)
  5. Copy the Public key for signature verification
  6. Save the webhook configuration
6

Fund Your Account

Ensure you have sufficient balance for payouts:

  1. Go to your Wise Home or Balances
  2. Click "Add money" on your desired currency
  3. Choose a funding method (bank transfer, card, etc.)
  4. Transfer funds to your Wise account
Tip: Keep balances in currencies you frequently pay out to minimize exchange rate fees.
7

Test in Sandbox (Recommended)

Test your integration before going live:

  1. Sign up for a sandbox account at sandbox.transferwise.tech
  2. Generate a sandbox API token following the same steps
  3. Select "Sandbox" environment when configuring
  4. Test payouts with sandbox credentials
  5. Switch to "Production" when ready to go live

Fee Information

Wise uses the mid-market exchange rate with transparent fees:

  • Exchange rate: Real mid-market rate with no hidden markup
  • Transfer fee: Varies by currency corridor (typically 0.3% - 2%)
  • Fee display: Exact fees shown before each transfer
  • Speed: Most transfers arrive within 1-2 business days

Use the Wise pricing calculator to check fees for specific corridors.
