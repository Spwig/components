Airwallex API Setup

Follow these steps to obtain your Airwallex API credentials for processing affiliate bank transfers.

1

Create an Airwallex Account

Visit airwallex.com and sign up for a business account.

Airwallex requires business verification. Have your business registration documents ready.

2

Complete Business Verification

After signing up, complete the KYB (Know Your Business) verification process:

  • Provide business registration details
  • Upload required identification documents
  • Verify beneficial owners

Verification typically takes 1-3 business days.

3

Access Developer Settings

Once verified, log in to your Airwallex account and navigate to:

Settings → Developer → API Keys

4

Generate API Credentials

Create a new API key set:

  1. Click Create API Key
  2. Select the appropriate permissions (Transfers, Beneficiaries)
  3. Copy your Client ID and API Key

Store your API Key securely - it will only be shown once.

5

Configure Webhooks (Recommended)

Set up webhooks to receive real-time transfer status updates:

  1. Go to Settings → Developer → Webhooks
  2. Click Add Endpoint
  3. Enter your webhook URL: {{ webhook_url }}
  4. Select these events:
    • transfer.ready_for_funding
    • transfer.batch_confirmed
    • transfer.completed
    • transfer.failed
    • transfer.returned
  5. Save and copy the Webhook Secret
6

Fund Your Account

Before processing payouts, ensure your Airwallex wallet has sufficient balance:

  • Go to Wallet in the dashboard
  • Add funds via bank transfer or linked account
  • Funds in the correct currency will be used for payouts
7

Test in Demo Mode

Airwallex provides a demo environment for testing:

  • Demo API Base: api-demo.airwallex.com
  • Use demo credentials to test transfers without real money
  • Switch to Production when ready to go live

Need Help? Visit Airwallex Payouts Documentation for detailed API information.
