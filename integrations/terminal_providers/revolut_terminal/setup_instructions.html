Revolut Terminal Setup

Revolut Terminal is a pure server-to-server integration. Push payment requests directly to Revolut Terminal and Revolut Reader devices — no terminal SDKs, drivers, or cables required.

Step 1: Get Your Revolut API Key

  1. Log into your Revolut Business Dashboard
  2. Navigate to Developer → API Keys
  3. Copy your Secret Key (starts with sk_live_ or sk_test_)
Note: This is the same secret key used for online payments. If you already have the Revolut payment provider configured, use the same key.

Step 2: Set Up a Location

  1. In Revolut Business, go to Merchant → Locations
  2. Create a location for your physical store
  3. Copy the Location ID (format: loc_...)
  4. Enter it in the Location ID field below
Important: Orders and terminals must be linked to the same location. The Location ID is required for terminal payments to work.

Step 3: Set Up Your Terminal Hardware

Revolut offers two hardware options:

Device Best For Connection
Revolut Terminal Counter-top retail, restaurants WiFi / Ethernet
Revolut Reader Mobile/portable, markets, pop-ups WiFi / Bluetooth
  1. Order your hardware from the Revolut Business store
  2. Power on the device and connect to WiFi
  3. Sign in with your Revolut Business credentials
  4. Set the terminal to Pay at Counter mode
  5. Assign it to the same location as configured above

Step 4: Enter Your Credentials Below

  1. Paste your Secret Key
  2. Select your Environment (Sandbox or Production)
  3. Enter your Location ID
  4. Click "Test Connection" to verify

Step 5: Testing Without Hardware

Sandbox Testing: Use the Sandbox environment to test the integration without physical hardware. Revolut provides simulated terminals in sandbox mode.
