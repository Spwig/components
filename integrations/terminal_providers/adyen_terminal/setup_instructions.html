Setting up Adyen Terminal

  1. Get an Adyen account — Sign up at adyen.com/signup
  2. Create an API key — In the Adyen Customer Area, go to Developers > API credentials and create a new API key with Terminal API and Management API permissions.
  3. Note your merchant account — Found under Settings > Merchant accounts in the Customer Area.
  4. Register your terminal — Order a terminal from Adyen and register it in the Customer Area under Point of Sale > Terminals.
  5. Connect your terminal — Power on your terminal and connect it to Wi-Fi. It will appear automatically in Spwig POS settings once online.
  6. Test the connection — Use the "Test Connection" button in the POS Terminal Provider settings to verify API connectivity.

Supported Terminals

  • S1F2L — Smart reader with large touchscreen (recommended)
  • P400 — Countertop payment terminal
  • S1F2 — Portable smart reader
  • S1E — Compact mobile terminal

Environment

Use the test environment during development. Switch to live for production — you'll need your live URL prefix from the Customer Area.
