Setting up SumUp Terminal

  1. Get a SumUp account — Sign up at sumup.com/signup
  2. Create an API key — In the SumUp Developer Dashboard, create an API key with payments and transactions.history scopes.
  3. Create an Affiliate Key — In the Developer Dashboard, go to Affiliate Keys and create one. This is required for Cloud API transactions.
  4. Note your Merchant Code — Found in your SumUp profile settings or Dashboard.
  5. Set up your Solo reader — Power on your SumUp Solo and follow the on-screen instructions to connect to WiFi.
  6. Pair the reader — The Solo displays a pairing code. Your reader will appear automatically in Spwig POS once paired.
  7. Test the connection — Use the "Test Connection" button in the POS Terminal Provider settings to verify API connectivity.

Supported Devices

  • SumUp Solo — Standalone card reader with touchscreen display and WiFi/SIM connectivity
  • SumUp Solo Lite — Compact card reader

Testing

SumUp provides a Virtual Solo for testing without physical hardware. Use a test API key (prefixed with sk_test_) during development. Note: a checkout amount of 11.00 in any currency will always fail — useful for testing error handling.
