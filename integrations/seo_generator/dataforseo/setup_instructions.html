Setting Up DataForSEO

1

Create a DataForSEO Account

Sign up at DataForSEO to get API access. New accounts receive a free trial balance.

Open DataForSEO Signup
2

Get Your API Credentials

In your DataForSEO dashboard, go to API Access. You'll need your login email and API password (this is different from your account password).

Your API password is auto-generated and shown in the API Access section of your dashboard. It is not the same as your account password.
3

Enter Credentials

Enter your login email and API password in the next step of this wizard to connect your account.
