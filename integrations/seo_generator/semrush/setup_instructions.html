Setting Up SEMrush

1

Create a SEMrush Account

Sign up at SEMrush to get API access. A paid plan with API units is required.

Open SEMrush Signup
2

Get Your API Key

In your SEMrush account, navigate to Subscription Info and find the API units section. Copy your API key from there.

API access requires a paid SEMrush plan. Check your available API units in the subscription settings.
3

Select Regional Database

Choose the regional database that best matches your target market. This determines which keyword data is used for SEO optimization. For example, if your store primarily targets US customers, select "United States".

Important Information

  • API Units: Each keyword request consumes API units. 10,000 units are included by default.
  • Regional Database: Select the database matching your target market for the most relevant keyword data.
