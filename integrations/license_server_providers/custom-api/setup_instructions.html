Custom API Setup

This option allows you to integrate with your own license server via REST API. Perfect for merchants with existing license management systems.

Requirements: Your license server must implement the required REST API endpoints. View API specification
  1. Ensure your license server is accessible via HTTPS
  2. Prepare your API base URL (e.g., https://licenses.yoursite.com/api)
  3. Prepare your authentication credentials (token, API key, or basic auth)
  4. Review the required API endpoints:
    • POST /licenses - Create license
    • GET /licenses/{key}/validate - Validate license
    • POST /licenses/{key}/activate - Activate device
    • POST /licenses/{key}/deactivate - Deactivate device
  5. Optional: Map your custom endpoints if they differ from the defaults

You'll need:

  • API Base URL - Your license API endpoint
  • Authentication Type - Bearer Token, API Key, Basic Auth, or Custom
  • Authentication Token/Key - Your API credentials
