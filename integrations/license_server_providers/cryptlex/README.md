# Cryptlex License Provider

**Version:** 1.0.0
**Author:** Spwig
**Type:** License Server Provider Component

## Overview

Cryptlex provides secure license management with advanced features like node-locked licenses, floating licenses, and offline activation.

## Features

- ✅ **Node-Locked Licenses** - Tied to specific devices
- ✅ **Floating Licenses** - Concurrent user licensing
- ✅ **Offline Activation** - Works without internet
- ✅ **Device Fingerprinting** - Secure device identification
- ✅ **License Policies** - Flexible license terms
- ✅ **Analytics** - Usage tracking and insights
- ✅ **Webhooks** - Real-time event notifications

## Setup

1. Sign up at [cryptlex.com](https://cryptlex.com)
2. Create a product in the dashboard
3. Generate a Personal Access Token (Profile → Access Tokens)
4. Note your Account ID and Product ID
5. Connect via Spwig wizard
