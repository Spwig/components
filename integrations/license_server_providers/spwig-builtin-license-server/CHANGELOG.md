# Changelog

All notable changes to the Spwig Built-in License Server provider will be documented in this file.

## [1.0.0] - 2026-01-21

### Added
- Initial release of Spwig Built-in License Server component
- Full license lifecycle management (create, validate, suspend, revoke)
- Device activation and deactivation support
- Offline validation token generation
- Real-time webhook integration
- Advanced analytics and reporting
- Usage tracking and metrics
- Trial license support
- Floating license support
- 99.9% uptime SLA guarantee
- Encrypted credential storage (Fernet encryption)
- Comprehensive error handling
- Connection testing endpoint
- Full integration with Spwig shop platform
