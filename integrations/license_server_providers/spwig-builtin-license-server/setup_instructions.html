Spwig Built-in License Server

Spwig's built-in license management server is fully-managed and requires no external setup!

  1. In the next step, you'll enter your Spwig Built-in License Server credentials
  2. If you don't have an account yet, sign up at: licenses.spwig.com
  3. After signing up, you'll receive your Account ID and API Key
Recommended: The built-in license server provides the easiest integration with full feature support.

Features

  • Fully Managed: No server setup, maintenance, or scaling required
  • Automatic Sync: Licenses sync automatically when orders are placed
  • Built-in Analytics: Track license usage, activations, and more
  • Offline Validation: Generate validation tokens for offline use
  • Device Management: Track and manage device activations
  • Webhooks: Real-time notifications for license events
  • 99.9% Uptime SLA: Enterprise-grade reliability
