Keygen.sh Setup

Keygen.sh is a popular license management platform that offers policy-based licensing with comprehensive device tracking and analytics.

  1. Sign up for a Keygen.sh account at keygen.sh
  2. Create a new product in your Keygen dashboard
  3. Create a policy for your product (defines license terms and limitations)
  4. Generate a Product API Token:
    • Go to Settings → Tokens
    • Click "Create Product Token"
    • Copy the generated token
  5. Note your Account ID from the URL: app.keygen.sh/accounts/YOUR_ACCOUNT_ID

You'll need:

  • Account ID - Found in your Keygen dashboard URL
  • Product API Token - Generated in Settings → Tokens
Product Mapping: After connecting, you'll need to map your Spwig products to Keygen policies in the provider configuration.

Features

  • Policy-Based Licensing: Define flexible license terms and limitations
  • Device Tracking: Monitor and manage device activations
  • Usage Analytics: Track license usage and activations
  • Webhooks: Real-time notifications for license events
  • Floating Licenses: Support for concurrent user licensing
  • Multi-Platform: SDKs for Windows, macOS, Linux, and more
