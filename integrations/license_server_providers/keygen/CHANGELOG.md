# Changelog

All notable changes to the Keygen.sh provider will be documented in this file.

## [1.0.0] - 2026-01-21

### Added
- Initial release of Keygen.sh license provider component
- Policy-based license management
- Device activation and deactivation
- License validation via Keygen API
- Machine/device tracking
- Webhook support for real-time events
- Floating license support
- Usage tracking and analytics
- Product-to-policy mapping configuration
- Sandbox environment support for testing
- Encrypted credential storage
- Comprehensive error handling
- Connection testing
