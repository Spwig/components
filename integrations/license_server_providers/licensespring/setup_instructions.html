License Spring Setup

License Spring provides comprehensive license management with support for floating licenses, consumption tracking, and trial licenses.

  1. Sign up for License Spring at licensespring.com
  2. Log in to your License Spring dashboard
  3. Create a new product
  4. Get your Management API Key:
    • Go to Account Settings → API Keys
    • Copy your Management API Key
  5. Get your Shared Key:
    • Go to Products → Your Product → SDK
    • Copy the Shared Key
  6. Note your Product Code from the product details

You'll need:

  • Management API Key - From Account Settings
  • Shared Key - From Product SDK settings
  • Product Code - Your product identifier

Features

  • Floating Licenses: Concurrent user licensing
  • Consumption Tracking: Monitor usage and consumption
  • Trial Licenses: Time-limited trials
  • Offline Activation: Works without internet
  • Multi-Platform: Windows, macOS, Linux, mobile
  • Analytics: Usage reports and insights
