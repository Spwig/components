# License Spring Provider

**Version:** 1.0.0
**Author:** Spwig
**Type:** License Server Provider Component

## Overview

License Spring provides comprehensive license management with advanced features like floating licenses, consumption tracking, and trial licenses.

## Features

- ✅ **Floating Licenses** - Concurrent user licensing
- ✅ **Consumption Tracking** - Monitor usage and consumption
- ✅ **Trial Licenses** - Time-limited trial support
- ✅ **Offline Activation** - Works without internet connection
- ✅ **Multi-Platform** - Windows, macOS, Linux, mobile support
- ✅ **Analytics** - Usage reports and insights
- ✅ **Webhooks** - Real-time event notifications

## Setup

1. Sign up at [licensespring.com](https://licensespring.com)
2. Create a product in the dashboard
3. Get your Management API Key (Account Settings → API Keys)
4. Get your Shared Key (Products → Your Product → SDK)
5. Note your Product Code
6. Connect via Spwig wizard
