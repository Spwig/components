Prerequisites

  1. A Facebook Page that you manage (not a personal profile)
  2. A Facebook Developer Account at developers.facebook.com
  3. A registered Facebook App with Pages API access

Step 1: Create a Facebook App

  1. Go to Facebook Developer Apps
  2. Click "Create App"
  3. Select "Business" as the app type
  4. Enter your app name and contact email
  5. Complete the app creation process

Step 2: Configure OAuth Settings

  1. In your app dashboard, go to App Settings > Basic
  2. Note your App ID and App Secret
  3. Add your store domain to App Domains
  4. Go to Facebook Login > Settings
  5. Add your OAuth redirect URI: {redirect_uri}
  6. Enable "Login with the JavaScript SDK" and "Web OAuth Login"

Step 3: Request Required Permissions

Your app needs these permissions approved:

  • pages_manage_posts - To create and publish posts to your Page
  • pages_read_engagement - To read Page insights and engagement data
  • pages_show_list - To list Pages you manage

Note: For development, you can use these permissions in Development Mode with your own Page. For production use with other users, submit for App Review.

Step 4: Get a Page Access Token

  1. Go to Graph API Explorer
  2. Select your app from the dropdown
  3. Click "Generate Access Token"
  4. Grant the required permissions when prompted
  5. Use the Access Token Debugger to exchange for a long-lived token

Step 5: Enter Credentials

In the next step, you'll need to enter:

  • Page Access Token - The long-lived token from Step 4
  • Page ID - Your Facebook Page's numeric ID (found in Page Settings > About)
Tip: Page Access Tokens expire after ~60 days. Enable automatic token refresh in your account settings to maintain connectivity.
