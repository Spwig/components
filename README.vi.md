## Tổng quan

Kho lưu trữ này là thư viện thành phần mã nguồn mở cho [Spwig](https://spwig.com), một nền tảng thương mại điện tử tự lưu trữ. Nó chứa ba loại thành phần mà cài đặt Spwig có thể sử dụng:

| | |
|--|--|
| **Giao diện** | Thiết kế cửa hàng sẵn dùng |
| **Tiện ích** | Trợ giúp giao diện quản trị (bảng chọn màu, trình chỉnh sửa gradient, chọn điểm tiêu điểm, v.v.) |
| **Tích hợp** | Nhà cung cấp thanh toán, vận chuyển, SMS, email, dịch thuật, SEO và mạng xã hội |

Mỗi thành phần được thiết kế để cài đặt và cập nhật thông qua trang quản trị Spwig — không cần viết mã — nhưng mã nguồn sống ở đây để bạn có thể kiểm tra, đóng góp hoặc fork.

## Cài đặt

### Thông qua trang quản trị Spwig (khuyên dùng)

Các thành phần được cài đặt chỉ với một cú nhấp chuột từ trang quản trị cửa hàng Spwig:

**Quản trị → Thị trường → Thành phần**

Thị trường hiển thị mọi thành phần trong kho lưu trữ này, cùng với các thành phần cao cấp và bên thứ ba. Nhấp vào **Cài đặt**, và Spwig sẽ tải thành phần, tích hợp vào trang web của bạn và áp dụng các di chuyển cần thiết.

### Cài đặt thủ công

Nếu bạn đang chạy cấu hình Spwig tùy chỉnh hoặc phát triển ngoại tuyến, bạn có thể cài đặt các thành phần theo cách thủ công:

```bash
git clone https://github.com/Spwig/components.git
cp -r components/themes/botanica /path/to/spwig/media/components/themes/
```

Sau đó, trong trang quản trị Spwig, chạy **Thị trường → Quét lại thành phần cục bộ**.

## Phát triển thành phần của riêng bạn

Xây dựng giao diện, tiện ích hoặc tích hợp mới? Bắt đầu với một trong các SDK của chúng tôi — mỗi cái là một khung nhỏ được tài liệu hóa tốt với nhà cung cấp mẫu bạn có thể fork:

| SDK | Để xây dựng |
|-----|-------------|
| [`theme-sdk`](https://github.com/Spwig/theme-sdk) | Giao diện với token, CSS quan trọng và hỗ trợ page builder |
| [`provider-sdks`](https://github.com/Spwig/provider-sdks) | Tích hợp thanh toán, vận chuyển, SMS, email và nhà cung cấp khác |
| [`headless-sdk`](https://github.com/Spwig/headless-sdk) | Cửa hàng headless gọi API Spwig |
| [`react`](https://github.com/Spwig/react) | React hooks và thành phần cho cửa hàng headless |

Mỗi thành phần trong kho lưu trữ này tuân theo cùng một định dạng manifest (`manifest.json`) để có thể được phát hiện bởi thị trường Spwig.

## Đóng góp

Đóng góp được hoan nghênh. Để giữ mọi thứ đơn giản, chúng tôi sử dụng **Developer Certificate of Origin (DCO)** thay vì CLA — chỉ cần thêm dòng ký tên vào commit của bạn:

```bash
git commit -s -m "Add support for X"
```

Vui lòng xem [CONTRIBUTING.md](CONTRIBUTING.md) để biết chi tiết về cách gửi PR, định dạng manifest và quy ước mã hóa.

**Ý tưởng đóng góp:**

- Giao diện mới cho các lĩnh vực chưa được phục vụ đầy đủ (ẩm thực, sức khỏe, giáo dục, bất động sản)
- Nhà cung cấp thanh toán hoặc vận chuyển bổ sung theo khu vực
- Sửa lỗi và cải tiến các thành phần hiện có
- Bản dịch của README và hướng dẫn thiết lập thành phần
