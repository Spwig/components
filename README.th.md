## ภาพรวม

รีโพซิทอรีนี้เป็นไลบรารีคอมโพเนนต์โอเพนซอร์สสำหรับ [Spwig](https://spwig.com) ซึ่งเป็นแพลตฟอร์มอีคอมเมิร์ซแบบโฮสต์เอง มีคอมโพเนนต์สามประเภทที่การติดตั้ง Spwig สามารถใช้ได้:

| | |
|--|--|
| **ธีม** | ดีไซน์หน้าร้านพร้อมใช้งาน |
| **เครื่องมือ** | ตัวช่วยสำหรับหน้าจัดการ (ตัวเลือกสี, ตัวแก้ไขไล่โทนสี, ตัวเลือกจุดโฟกัส ฯลฯ) |
| **การเชื่อมต่อ** | ผู้ให้บริการชำระเงิน, จัดส่ง, SMS, อีเมล, การแปล, SEO และโซเชียลมีเดีย |

คอมโพเนนต์แต่ละตัวได้รับการออกแบบมาให้ติดตั้งและอัปเดตผ่านหน้าจัดการของ Spwig โดยไม่ต้องเขียนโค้ด แต่ซอร์สโค้ดอยู่ที่นี่เพื่อให้คุณตรวจสอบ ร่วมพัฒนา หรือ fork ได้

## การติดตั้ง

### ผ่านหน้าจัดการ Spwig (แนะนำ)

คอมโพเนนต์ติดตั้งด้วยการคลิกครั้งเดียวจากหน้าจัดการร้านค้า Spwig ของคุณ:

**Admin → Marketplace → คอมโพเนนต์**

Marketplace แสดงคอมโพเนนต์ทั้งหมดในรีโพซิทอรีนี้ พร้อมกับคอมโพเนนต์พรีเมียมและจากบุคคลที่สาม คลิก **ติดตั้ง** และ Spwig จะดึงคอมโพเนนต์ เชื่อมเข้ากับไซต์ของคุณ และใช้การย้ายข้อมูลที่จำเป็น

### การติดตั้งแบบแมนนวล

หากคุณใช้การตั้งค่า Spwig ที่ปรับแต่งหรือพัฒนาแบบออฟไลน์ คุณสามารถติดตั้งคอมโพเนนต์ด้วยตนเอง:

```bash
git clone https://github.com/Spwig/components.git
cp -r components/themes/botanica /path/to/spwig/media/components/themes/
```

จากนั้นในหน้าจัดการ Spwig ให้เรียกใช้ **Marketplace → สแกนคอมโพเนนต์ท้องถิ่นอีกครั้ง**

## พัฒนาคอมโพเนนต์ของคุณเอง

ต้องการสร้างธีม เครื่องมือ หรือการเชื่อมต่อใหม่? เริ่มต้นด้วย SDK ตัวใดตัวหนึ่งของเรา แต่ละตัวเป็นเฟรมเวิร์กขนาดเล็กที่มีเอกสารครบถ้วน พร้อมผู้ให้บริการตัวอย่างที่คุณ fork ได้:

| SDK | สำหรับสร้าง |
|-----|-------------|
| [`theme-sdk`](https://github.com/Spwig/theme-sdk) | ธีมพร้อม token, critical CSS และการรองรับ page builder |
| [`provider-sdks`](https://github.com/Spwig/provider-sdks) | การเชื่อมต่อการชำระเงิน จัดส่ง SMS อีเมล และผู้ให้บริการอื่น ๆ |
| [`headless-sdk`](https://github.com/Spwig/headless-sdk) | หน้าร้าน headless ที่เรียกใช้ Spwig API |
| [`react`](https://github.com/Spwig/react) | React hooks และคอมโพเนนต์สำหรับหน้าร้าน headless |

คอมโพเนนต์ทุกตัวในรีโพซิทอรีนี้ใช้รูปแบบ manifest เดียวกัน (`manifest.json`) เพื่อให้ Spwig marketplace ค้นพบได้

## การมีส่วนร่วม

ยินดีต้อนรับการมีส่วนร่วม เพื่อให้เรียบง่าย เราใช้ **Developer Certificate of Origin (DCO)** แทน CLA เพียงเพิ่มบรรทัดลงชื่อในคอมมิตของคุณ:

```bash
git commit -s -m "Add support for X"
```

โปรดดู [CONTRIBUTING.md](CONTRIBUTING.md) สำหรับรายละเอียดเกี่ยวกับการส่ง PR รูปแบบ manifest และแบบแผนการเขียนโค้ด

**แนวคิดในการมีส่วนร่วม:**

- ธีมใหม่สำหรับหมวดหมู่ที่ยังไม่ได้รับการรองรับ (อาหารและเครื่องดื่ม สุขภาพ การศึกษา อสังหาริมทรัพย์)
- ผู้ให้บริการชำระเงินหรือจัดส่งเพิ่มเติมตามภูมิภาค
- แก้ไขข้อบกพร่องและปรับปรุงคอมโพเนนต์ที่มีอยู่
- การแปล README และคำแนะนำการตั้งค่าคอมโพเนนต์
