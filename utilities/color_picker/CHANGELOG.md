# Changelog - Color Picker Utility

## [1.0.1] - 2025-10-09

### Changed
- **UI Improvement**: Simplified trigger button to show only the palette icon
  - Icon color now reflects the selected color
  - Removed separate color swatch preview
  - Icon displays in default color when no color is selected or when transparent
  - Cleaner, more minimalist appearance

### Technical
- Removed `.color-preview` element and associated CSS
- Updated trigger button to dynamically color the palette icon
- Improved color validation for transparent values

## [1.0.0] - 2025-10-08

### Added
- Initial release of Color Picker utility
- Support for multiple color formats (hex, rgb, rgba, hsl, hsla)
- Opacity/alpha channel control
- Preset color swatches
- Recent colors history
- Live preview integration
- Eyedropper tool support
- Responsive design
- Dark mode support
- RTL language support
- Accessibility features
