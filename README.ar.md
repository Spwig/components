## نظرة عامة

هذا المستودع هو مكتبة المكونات مفتوحة المصدر لـ [Spwig](https://spwig.com)، وهي منصة تجارة إلكترونية ذاتية الاستضافة. يحتوي على ثلاثة أنواع من المكونات التي يمكن أن يستخدمها تثبيت Spwig:

| | |
|--|--|
| **القوالب** | تصاميم واجهة متجر جاهزة للاستخدام |
| **الأدوات** | مساعدات واجهة المسؤول (منتقي الألوان، محرر التدرجات، منتقي نقطة التركيز، إلخ) |
| **التكاملات** | مزودو الدفع والشحن والرسائل النصية والبريد الإلكتروني والترجمة والسيو والشبكات الاجتماعية |

كل مكون مصمم ليتم تثبيته وتحديثه من خلال إدارة Spwig — دون أي كود — لكن المصدر يعيش هنا حتى تتمكن من فحصه أو المساهمة أو عمل fork.

## التثبيت

### عبر إدارة Spwig (موصى به)

تُثبت المكونات بنقرة واحدة من إدارة متجر Spwig الخاص بك:

**الإدارة ← المتجر ← المكونات**

يعرض المتجر جميع المكونات في هذا المستودع، بالإضافة إلى المكونات المتميزة ومكونات الأطراف الثالثة. انقر فوق **تثبيت**، وسيقوم Spwig بجلب المكون ودمجه في موقعك وتطبيق أي هجرات مطلوبة.

### التثبيت اليدوي

إذا كنت تشغل إعدادًا مخصصًا لـ Spwig أو تطور دون اتصال، يمكنك تثبيت المكونات يدويًا:

</div>

```bash
git clone https://github.com/Spwig/components.git
cp -r components/themes/botanica /path/to/spwig/media/components/themes/
```

<div dir="rtl">

ثم، في إدارة Spwig، شغل **المتجر ← إعادة فحص المكونات المحلية**.

## تطوير مكونك الخاص

هل تبني قالبًا أو أداة أو تكاملًا جديدًا؟ ابدأ بأحد SDKs الخاصة بنا — كل منها إطار عمل صغير موثق جيدًا مع مزود مثال يمكنك عمل fork له:

| SDK | لبناء |
|-----|-------|
| [`theme-sdk`](https://github.com/Spwig/theme-sdk) | قوالب مع الرموز و CSS الحرج ودعم منشئ الصفحات |
| [`provider-sdks`](https://github.com/Spwig/provider-sdks) | تكاملات الدفع والشحن والرسائل النصية والبريد الإلكتروني ومزودين آخرين |
| [`headless-sdk`](https://github.com/Spwig/headless-sdk) | واجهات متاجر headless تستدعي واجهة برمجة تطبيقات Spwig |
| [`react`](https://github.com/Spwig/react) | React hooks ومكونات لواجهات المتاجر headless |

يتبع كل مكون في هذا المستودع نفس تنسيق manifest (`manifest.json`) ليتمكن متجر Spwig من اكتشافه.
